## Getting started (OSX)

These instructions refer to **OSX** with **Mathematica 11.0** and **Processing 3.4**.

### Install Wolfing

1. Clone this repo to your sketchbook folder

		cd Documents/Processing/sketchbook
		clone https://github.com/bitcraftlab/wolfing.git
		
2.	Enter the repo

		cd wolfing
		
2. Add JLink files:

		make link
